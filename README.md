
# Co Ways

Co Ways merupakan platform Beta untuk mendengarkan music dan pengerjaan hanya dalam waktu 1 minggu. Bisa dikembangkan lagi untuk UI/UX yang nyaman. Ini merupakan RestFull API dari platform Co Ways.
